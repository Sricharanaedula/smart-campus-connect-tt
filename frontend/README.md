# Frontend (static)

This is a minimal static frontend for local development. Files are in this folder and also copied into the backend static folder so Spring Boot serves them directly.

To iterate locally:

1. Edit `index.html`, `styles.css`, `app.js` in `frontend/`.
2. Copy files into backend static folder (automate later) or run the backend which already serves the included assets.

No build step required for this simple static site.
